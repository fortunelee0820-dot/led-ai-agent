# Wisva AI Sales Agent

AI-powered workflow for LED manufacturing and export sales automation.

## Features

- AI email generation
- Customer inquiry analysis
- LED product recommendation
- MOQ & certification knowledge retrieval
- Factory workflow coordination
- Long-context customer follow-up
- AI-assisted quotation generation

## Workflow

Customer Inquiry
↓
AI Requirement Analysis
↓
Product Matching
↓
Knowledge Base Retrieval
↓
Quotation Suggestion
↓
English Email Generation
↓
Sales Confirmation
↓
Long-term Customer Follow-up

## Tech Stack

- LLM
- AI Agent Workflow
- Prompt Engineering
- Knowledge Base
- RAG
- Long Context Processing

## Use Case

Designed for LED strip manufacturing export business scenarios, including:
- COB strips
- Neon strips
- Power supplies
- OEM/ODM projects
